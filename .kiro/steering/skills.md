# Memoria Skills

This project organizes knowledge as skills in the `skills/` directory. Each skill has a `SKILL.md` with frontmatter and actionable instructions.

| Skill | When to use |
|-------|-------------|
| [Architecture](../../skills/architecture/SKILL.md) | Navigating or modifying Memoria code |
| [API Reference](../../skills/api-reference/SKILL.md) | Calling or implementing REST endpoints |
| [Deployment](../../skills/deployment/SKILL.md) | Deploying or configuring Memoria |
| [Plugin Development](../../skills/plugin-development/SKILL.md) | Creating or managing governance plugins |
| [Setup](../../skills/setup/SKILL.md) | Helping users install Memoria |
| [Release](../../skills/release/SKILL.md) | Publishing a new version |
| [Local Embedding](../../skills/local-embedding/SKILL.md) | Setting up offline embedding |

Load the relevant skill before starting a task.
